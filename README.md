## 헬다이버즈2 스트라타젬 보조 기구
* **즉각적인 민주주의 전파**<br>설정창에서 낭비할 시간은 없습니다. 실행 즉시 게임 설정을 감지하여 귀하를 전장으로 사출합니다.

* **영구적인 화력 지원**<br>본부 통신이 끊겨도 `stratagems.json` 직접 수정을 통해 신규 무기를 보강하는 것은 합법적인 전술적 자유입니다.

* **단 한 번의 보급 요청**<br>복잡한 커맨드는 신병에게나 맡기십시오. 단축키 하나로 궤도 타격부터 중화기 보급까지 손끝에서 해결합니다.

* **전술적 라디얼 UI**<br>난전 속에서 메뉴를 찾는 건 반역 행위입니다. 직관적인 원형 UI로 원하는 전략 자산을 단번에 식별하고 투하하십시오.

* [보조 기구 장착하기 (Download)](https://github.com/ChubbyMaru/HD2-Stratagem-Tool/releases/latest)

## 단축키 기본 값
* **증원 :** `마우스 버튼1(뒤로 가기)`
* **오버레이 :** `마우스 휠 클릭`
* **인게임 선택 :** `F1`
* **슬롯 :** `F6` ~ `F12`
